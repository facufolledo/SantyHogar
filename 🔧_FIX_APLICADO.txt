❌ PROBLEMA:
   ERROR 42601: syntax error at or near "UNIQUE"
   CREATE INDEX idx_clientes_email ON clientes(email UNIQUE)
   
   PostgreSQL no reconoce esta sintaxis

════════════════════════════════════════════════════════════════════════════════

✅ SOLUCIÓN:
   CREATE UNIQUE INDEX idx_clientes_email ON clientes(email)
   
   Sintaxis correcta para PostgreSQL/Supabase
